## 📌 About This Repository

This repository contains the source code for Clanker.

The code is open source so the community can:

- 🐛 Report bugs
- 💡 Suggest improvements
- 🔧 Contribute fixes and features
- 📖 Learn from the project

This repository is **not intended for people to create their own hosted versions of Clanker**.

Clanker is a publicly hosted bot, and we want contributions to improve the official bot rather than separate self-hosted copies.

## ⚠️ Self Hosting

Self-hosting Clanker is **not officially supported**.

The code may depend on:
- private configuration
- external services
- internal systems
- infrastructure that is not included in this repository

You are welcome to look through the code and contribute, but getting your own instance running is not guaranteed.
